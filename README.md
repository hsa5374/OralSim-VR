# OralSim ‚Äì VR Dental Scaling Training Simulator

**OralSim** is a Unity-based Virtual Reality dental training simulator designed for the Meta Quest headset.  
The project teaches dental students essential skills such as oral inspection, plaque identification, and ultrasonic scaling using an immersive VR environment.

This simulation provides a sustainable, repeatable, and interactive alternative to physical dental practice models.

## üöÄ Features
- Fully immersive VR dental operatory scene
- Realistic patient model with dirty teeth
- Tool interactions: mouth mirror, explorer, ultrasonic scaler, suction, and more
- Wrong-tool detection with voice alerts  
- Guided instructions from an animated virtual dental instructor
- Real-time plaque removal with particle & sound effects
- Practice mode with progress bar & scoring
- User selection system + session management
- Unity XR + Meta Quest SDK integration
